# cbr_data_app

## Описание проекта

**Мобильное приложение** для работы с экономическими данными Центрального Банка России, предоставляющее пользователям удобный доступ к статистической информации через интуитивно понятный интерфейс.

Приложение позволяет:
* Получать и просматривать **списки публикаций** с экономической информацией
* Выбирать и **анализировать датасеты** по интересующим темам
* Определять доступный временной диапазон данных для выбранного датасета
* Запрашивать и отображать **статистические данные** за выбранный период
* Настраивать параметры запроса, включая использование расширенного формата данных

**Интерфейс приложения** включает:
* Панель загрузки данных с индикатором прогресса
* Секции для отображения категорий, датасетов и временных параметров
* Форму настройки параметров запроса
* Табличное представление полученных данных

**Система обработки ошибок** обеспечивает корректное отображение сообщений об ошибках и позволяет пользователю оперативно исправлять некорректные входные данные.
